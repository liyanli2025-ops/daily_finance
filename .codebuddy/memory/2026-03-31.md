# 2026-03-31 工作日志

## 新增市场数据增强模块

在 `market_data_service.py` 中新增了三大数据采集模块：

### 1. 概念板块排行 (`ConceptSectorData`)
- 使用 `ak.stock_board_concept_name_em` 获取东方财富概念板块行情
- 按涨跌幅排序取Top10
- 新增方法：`_get_concept_sectors()`

### 2. 连板强势股 (`ConsecutiveLimitStock`)
- 使用 `ak.stock_zt_pool_em` 涨停股票池接口，筛选连板>=2天的股票
- 按连板天数降序排列
- 新增方法：`_get_consecutive_limit_stocks()`

### 3. 技术信号机会股 (`TechSignalStock`)
- **路径A**：同花顺技术选股接口（量价齐升、连续上涨、创新高、持续放量）
  - 方法：`_get_ths_tech_stocks()`
- **路径B**：自研多因子技术信号扫描（初筛全A股→Top20拉K线→8大信号打分）
  - 方法：`_scan_tech_signals()`
  - 信号包括：MACD金叉、均线多头排列、突破20日均线、KDJ金叉、RSI超卖反弹、布林带突破中轨、放量、连续上涨
  - 至少触发2个信号才入选
- 合并方法：`_get_tech_signal_stocks()`（两路径合并去重，双路径共振加分）

### 关于播客
经分析，新增数据模块**不需要修改播客代码**。原因：
- 播客脚本由AI二次创作（不是照搬数据）
- 报告正文被 `_extract_key_paragraphs` 截取到1500字
- 播客prompt有"不要有模块感"、"一条主线"等防无聊约束
- 数据越丰富，AI 创作素材越好
