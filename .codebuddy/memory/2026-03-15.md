### 技术要点
- 使用 RSSHub `/wechat/mp/articles/{biz}` 路由采集公众号
- 支持多 RSSHub 实例自动切换（rssforever.com → rsshub.app → rss.shab.fun）
- 并发采集限制 3 个信号量，避免频繁请求
- 采集失败有计数器，连续失败会在 UI 提示
- 公众号源定位为"锦上添花"辅助源，采集失败不影响整体流程
