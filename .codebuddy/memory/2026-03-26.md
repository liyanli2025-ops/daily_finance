## WeWe-RSS 替换 RSSHub 作为公众号文章数据源

- **原因**：RSSHub 公共实例的微信公众号路由全部失效，被微信反爬封了
- **方案**：在服务器上用 Docker 部署 WeWe-RSS（基于微信读书接口，稳定可靠）
- **代码改造**：`wechat_service.py` 新增 WeWe-RSS 采集逻辑，优先使用 `http://localhost:4000`，不可用时降级到 RSSHub
- **WeWe-RSS 地址**：`http://localhost:4000`，feed API：`/feeds/all.json`
- **部署命令**：`docker run -d --name wewe-rss -p 4000:4000 -e DATABASE_TYPE=sqlite -e AUTH_CODE=xxx -v /var/www/wewe-rss-data:/app/data cooderl/wewe-rss-sqlite:latest`
- **注意**：WeWe-RSS 需要用微信扫码登录微信读书账号，然后在管理面板中订阅公众号
